A browser-based OSCAL workbench that supports the full workflow from profile and SSP creation through assessment and POA&M remediation, with integrated viewer functionality for browsing the catalog. The tools run as HTML/JS applications, support AI-assisted explanations, audit findings, and remediation planning, and can export OSCAL artifacts such as profiles, SSPs, AP/AR files, and POA&M documents.
