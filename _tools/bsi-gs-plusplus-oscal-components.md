A Python- and AI-based framework for generating OSCAL component definitions for the transition from BSI IT-Grundschutz Edition 2023 to the newer Grundschutz++ model. It supports strict 1:1 mapping from legacy Bausteine and Anforderungen to G++ target objects and controls, then enriches the resulting components with implementation details, maturity levels, and classifications.
